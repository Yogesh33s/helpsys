Smart Community Help System
Overview

Smart Community Help System is a beginner-friendly full-stack DevOps project developed using Java Spring Boot, MongoDB, Docker, Maven, GitHub Actions, and Jenkins.

The platform is designed for NGOs and public helping organizations to manage community help requests efficiently. The system helps organizations create requests, monitor completion status, track timestamps, and coordinate community support activities.

The project demonstrates modern DevOps concepts including Docker containerization, CI/CD workflows, Maven build automation, GitHub Actions integration, and Jenkins deployment pipelines.

How to Run the Project
Prerequisites
Docker Desktop installed and running
Step 1: Clone Repository
git clone https://github.com/Yogesh33s/helpsys.git
Step 2: Open Project Folder
cd helpsys
Step 3: Run Docker Containers
docker-compose up
Step 4: Open Browser

Frontend:
http://localhost:3000?

Backend API:
http://localhost:8080/api/requests?

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Java Spring Boot
Database: MongoDB
Build Tool: Maven
Containerization: Docker
CI/CD: GitHub Actions & Jenkins
